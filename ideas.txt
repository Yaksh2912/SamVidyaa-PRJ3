Ideas for Future Features - SamVidyaa 🚀

1. Real-Time Web Sockets (Notifications) 🔔
   - Implement Socket.io to provide live updates without needing page refreshes.
   - Notify students instantly when a peer requests collaboration, or when a teacher releases a new handout / grades an assignment.

2. Code Plagiarism Detection 🔍
   - Implement a lightweight AST (Abstract Syntax Tree) comparison tool for submitted code.
   - Flag suspiciously similar code submissions to teachers to ensure the integrity of lab assignments while still encouraging healthy collaboration.

3. Advanced Teacher Analytics Dashboard 📊
   - Enhance the `getTeacherStats` API and integrate a charting library like Recharts or Chart.js on the frontend.
   - Provide teachers with a heat map of difficult tasks, a bell curve of class grades, and a visual leaderboard snapshot of top-performing vs at-risk students.

4. Badges & Achievement System 🏆
   - Take gamification beyond just a "Reward Shop" and "Points."
   - Add dynamic achievements (e.g., "First Submission," "Bug Squasher," "Streak of 5 Days") that are permanently displayed on the student's profile for peers to see.
